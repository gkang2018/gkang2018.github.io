VirtualHeap

About the Project

VirtualHeap is a dynamic memory allocator in C that completes Malloc, Free, VMalloc, VFree requests. VMalloc/VFree use a redirection table to perform compaction which minimizes fragmentation. It also incorporates a HeapChecker to allow user to see current memory usage at runtime.
