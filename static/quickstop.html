QuickStop App

About the Project

QuickStop is a Flask web application that tries to find the most efficient method of transportation for the user throughout the city of Boston. The metrics of efficiency include price, distance traveled, and expected arrival time. The application originally used the Uber, Lyft, MBTA, and Google Maps API. However, since the Uber API is currently deprecated, the application only supports the latter three services. This project was designed at HackHarvard and has now become a personal project I intent to complete.
