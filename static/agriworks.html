Agriworks Web Application

About the Project

Agriworks is a web application that looks to help manage and understand agricultural resource consumption in India. The application inputs resource datasets and uses Python pandas and several other tools to create statistical interpretations of the data. These data models can be used by researchers, policymakers, non-profits, and citizens. The front-end is developed with VueJS and for the backend, we decided to implement it with Flask and MongoDB.
