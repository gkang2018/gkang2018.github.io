NodPay Mobile Application

About the Project

NodPay is a React Native mobile application with a NodeJS / Express back-end that attempts to revolutionize global transactions by providing users an opportunity to trade money in a secure, efficient, and interactive manner, making use of a social feed to see what all your friends are purchasing!
