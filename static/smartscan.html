SmartScan App

About the Project

SmartScan is an iOS application for the Wellesley Cancer Prevention Project that scans barcodes on food products and attempts to determine whether any ingredients are potentially carcinogenic. The application doesn't offer medical advice but rather cautions against consumption. Mobile application communicates with Flask server to determine whether the scanned barcode contains any such ingredients.
